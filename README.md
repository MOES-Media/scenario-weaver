## Introduction

**scenario-weaver** is an open source Java library for scenario-based testing.  
It enables developers to define, execute, and validate complex test scenarios with clear separation of context, actions, stubs, and assertions.  
The project is designed to be extensible, maintainable, and easy to integrate into modern CI/CD pipelines.  
Whether youâ€™re testing business logic, integrations, or APIs, scenario-weaver-core helps you automate and organize your tests for reliability and clarity.

Key features include:
- Modular scenario definition
- Pluggable providers for context, payload, stubs, actions, and assertions
- Integration with popular test frameworks and Spring framework
